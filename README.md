# HR-FEC

## Goal
This app is a single product page for an ecommerce website.
It contains:
- product details
- recommendations (related products, what goes well together)
- Q and A
- Ratings and Reviews

## Dependecies
- React
- React Router
- Axios
- Bootstrap CSS
- AWS s3

## Dev-dependencies
- Webpack
- Babel
- React Testing Library
- Jest
